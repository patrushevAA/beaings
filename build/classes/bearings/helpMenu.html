Инструкция по работе с программой


1. Для начала работы с программой подключите базу данных в формате .txt через пункт меню "Файл -> Открыть"

2. Во вкладке поиск введите необходимые Вам параметры подшпников. Затем проверьте правильность заполнения полей поиска через пункт меню "Фунции -> Проверить". При возникновении ошибок исправьте параметры поиска и произведите повторную проверку.

3. Нажмите кнопку "Поиск" или пункт меню "Функции -> Поиск" для поиска элементов в базе данных по указанным параметрам. В случае отстутствия элементов с указанными параментами программа выдаст предупреждение.

4. После проведения поиска постройте таблицу с результатами. Для этого выберите пункт меню "Функции -> Построить таблицу".

5. Для изменения значений "Дата поставки" и "Количество" двойным щелчом мыши выберите строку для изменения. В появившейся форме измените значение соответствующих параметров и нажмите кнопку "ОК". Для сброса введенных Ва значений нажмите кнопку "Сброс", для сохранения изменений кнопку "Сохранить".

6. После построения таблицы есть возможность увидеть остатки по подшипникам в графическом представлении. Для этого выберите пункт меню "Функции -> Построить график".

7. Для печати графика выберите пункт меню "Функции -> Напечатать график".

8. Для сохранения изменений в базе данных выберите пункт меню "Файл -> Сохранить".
